# Flask User Authentication System

## Overview

A Flask web application that demonstrates user registration, login authentication, session management, and SQLite database integration.

## Features

- User Registration
- User Login
- Dashboard Page
- SQLite Database Storage
- Flask-SQLAlchemy Integration
- Simple Web Interface

## Technologies Used

- Python
- Flask
- Flask-SQLAlchemy
- SQLite
- HTML

## Project Structure

app.py
templates/
├── login.html
├── register.html
└── dashboard.html

instance/
└── database.db

## Installation

```bash
pip install flask flask-sqlalchemy
```

## Run

```bash
python app.py
```
